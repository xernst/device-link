# Development Workflow — Right Brain

## Creative Development Pipeline

1. **Research First** (mandatory)
   - Use **researcher** agent for context
   - Search for prior art and existing solutions
   - Understand the landscape before designing

2. **Define Requirements**
   - Use **prd-creator** agent for product specs
   - User stories with acceptance criteria
   - Non-functional requirements identified

3. **Design Architecture**
   - Use **architect** agent for system design
   - Trade-off analysis documented
   - ADRs for significant decisions

4. **Plan Implementation**
   - Use **planner** agent to create step-by-step plan
   - Break into independently deliverable phases
   - Identify dependencies and risks

5. **Document**
   - Use **doc-updater** agent
   - Keep documentation in sync with decisions
   - Architecture diagrams, API docs

6. **Communicate**
   - Use **chief-of-staff** for stakeholder updates
   - Status reports, design proposals
